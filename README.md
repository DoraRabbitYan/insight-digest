# 💡 insight-digest

> 将长内容深度提炼为短文+长文两种格式，洞察优于信息。

## 一句话定位

长内容精华提炼引擎 — 播客、视频、文章深度分析后输出社交短文 + 叙事长文，核心是"非共识洞察"而非简单摘要。

## ✨ 功能点

- 🎯 **四阶段深度分析** — 提取 50+ 观点 → 筛选非共识洞察 → 提炼叙事 → 精选 10-15 条精华
- 📝 **短文输出** — 300-800 字社交平台格式，编号 emoji 列表，每条有完整逻辑链条
- 📖 **长文输出** — 2000-3500+ 字叙事文章，有故事弧线和分析深度
- 🗣 **双风格支持** — 对话式访谈（Style B，保留"主持人：""嘉宾："）和叙事性文章（Style A）
- ✅ **内置质检** — 每条洞察必须通过"非显而易见"测试——有见识的读者是否已经知道？

## 📦 安装

```bash
# 方式一：vskills 一键安装
vskills install insight-digest

# 方式二：Git 克隆
git clone https://github.com/dora/insight-digest.git
```

## 🚀 快速开始

```
"帮我总结这个播客 https://www.youtube.com/watch?v=xxx"
"把这篇长访谈转成短文+长文"
"这个视频讲了什么？提炼核心观点"
```

## 🧬 特色模块：四阶段深度分析法

| 阶段 | 目标 | 关键动作 |
|------|------|----------|
| Stage 1 | 全量提取 | 通读全文，提取 **50+** 观点（含隐含信念、决策逻辑、反例） |
| Stage 2 | 筛选非共识 | 标记反直觉、私下表达、冷知识、心智模型、二阶洞察 |
| Stage 3 | 提炼叙事 | 识别核心故事线、金句、转折点、戏剧性元素 |
| Stage 4 | 精选输出 | 定稿 10-15 条最深洞察，短文直接用，长文以此为基础展开 |

- 🎧 **播客笔记整理** — 2 小时播客 → 10 分钟读完所有精华观点

- 📚 **长文精读提炼** — 万字长文 → 一张卡片看完作者真正想说的
